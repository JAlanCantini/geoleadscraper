## What is this tool?
Geoleadscraper acts as a browser extension for Google Chrome. It reads information from digital maps and saves the data to your computer. You use this tool to gather business names, addresses, phone numbers, websites, and email addresses. It works with Google Maps, Yandex Maps, and 2GIS. You do not need an account or an API key to use it. It creates files that open in Excel or other spreadsheet software, which saves you time on manual data entry.

## ⚙️ Requirements
You need a computer running Windows 10 or Windows 11. You must have the Google Chrome web browser installed. This extension works within the Chrome environment. You do not need to install other software or manage database servers. An active internet connection allows the extension to reach the map services and download the information.

## 🛠️ Setting up the extension
Chrome requires a specific step to load extensions that do not come from the official web store.

1. Open Google Chrome.
2. Type chrome://extensions/ in the address bar and press Enter.
3. Look for the Developer mode switch in the top right corner.
4. Click the switch to turn it on.
5. Click the Load unpacked button on the top left.
6. Select the folder where you placed the extension files.
7. Chrome adds the extension to your browser immediately.

## 🕵️ How to use the tool
1. Click the puzzle icon in the top right corner of Chrome.
2. Select Geoleadscraper from the list.
3. Pin the extension to your toolbar for easier access.
4. Open Google Maps, Yandex Maps, or 2GIS in a new tab.
5. Search for a business category in your target city.
6. Click the extension icon once the map shows your results.
7. Use the interface to select the data fields you need.
8. Click the Start button.
9. Watch as the tool collects information from the listings.
10. Wait for the progress bar to finish.

The extension compiles the gathered information once the scan ends.

2. Choose where to save the file on your computer.
3. Open the file using Microsoft Excel, Google Sheets, or any CSV reader.
4. View your gathered leads in a structured table.

## 💡 Best practices
Focus on narrow search queries to get the best results. Examples include "Plumbers in Austin" or "Coffee shops in Seattle." Large cities often contain thousands of results. Use map filters to refine your search before you start the extension. This improves the speed of your data collection. Keep your browser window open while the extension works. Avoid navigating away from the map page during the scan.

## 🔧 Frequently asked questions
Do I need to pay for this?
No. This tool is free and open-source.

Does it work with other browsers?
It requires Chromium-based browsers. It works reliably in Google Chrome and Microsoft Edge.

How does it handle duplicate listings?
The extension logic ignores duplicate entries automatically to keep your list clean.

Is my data private?
Yes. The extension performs all work locally within your browser. It does not send your data to external servers.

Can I pause a scan?
Yes. Use the pause button in the extension interface to stop and resume the process whenever you need.

What happens if the map page refreshes?
The extension keeps track of the data it already collected. Refreshing the page might disrupt the current session, so you may need to restart the search.

Does it gather emails?
Yes. If a business website lists an email address, the extension attempts to find and save it.

## 🛡️ Privacy and safety
This tool respects your privacy. It does not track your browsing history. It does not require login information for map services. It connects only to the map websites you visit to pull the public business information displayed on your screen. You maintain total control over your downloaded CSV files at all times.
